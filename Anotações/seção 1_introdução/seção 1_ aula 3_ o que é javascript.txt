O que é JavaScript

* É uma linguagem de programação de alto nível;
* Originalmente chamada de LiveScript;
* Porém recebeu o nome de JavaScript por causa da grande fama do Java;
* JavaScript = JS = Vanilla JavaScript = ECMAScript;
* Criada para deixar as páginas web vivas;

* Js é Just-in-time compiled
// é executada em tempo real, ou seja, não existe um delay de tempo para compilação.

* Orientada a objetos;
* Criada em 1995;
