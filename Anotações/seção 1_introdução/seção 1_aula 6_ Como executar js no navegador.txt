* Como executar JS no navegador por arquivo

Como linkar o arquivo js externo no documento html: linkar dentro da tag <head> com <script src=""> exemplo: 

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <title>Introdução</title>
    <script src="script.js"></script>
</head>
<body>
    <h1>Este é o HTML correto</h1>
    <script>
        console.log("Hello world JavaScript!");
    </script>
</body>
</html>
